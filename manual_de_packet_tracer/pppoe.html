10. PPPOE

Configuracion del servidor pppoe.

En este ejemplo se utiliza una red de clase B:

Paso 1: Configurar el Router (Servidor PPPoE)

  1. Acceder al Router: Haz doble clic en el router y abre la CLI (Interfaz de Línea de Comandos).
  2. Configuración Básica:
    • Configura la interfaz de red con una dirección IP. Por ejemplo:
enable
configure terminal
interface GigabitEthernet0/0/0
ip address 172.16.0.1 255.255.0.0
no shutdown
exit
  1. Crear el Pool de Direcciones IP:
    • Crea un Conjunto \ Rango de direcciones IP para los clientes PPPoE:
ip local pool pppoepool 172.16.128.1 172.16.128.254
  1. Configurar el Grupo BBA:
    • Configura el grupo PPPoE (BBA) y asocia una plantilla virtual:
bba-group pppoe global
virtual-template 1
exit
  1. Crear la Plantilla Virtual:
    • Crea y configura la plantilla virtual para la conexión PPPoE:
interface Virtual-Template1
ip unnumbered GigabitEthernet0/0/0
peer default ip address pool pppoepool
ppp authentication pap
exit
  1. Configurar Usuario y Contraseña:
    • Configura las credenciales del cliente PPPoE:
username [username] password [password]

Sustituye [username] y [password] con las credenciales solicitadas en el examen o las que tu quieras.

  1. Activar PPPoE en la Interfaz:
    • Activa PPPoE en la interfaz Ethernet del router:
interface GigabitEthernet0/0/0
pppoe enable group global
exit

Configuración en PC - Ordenador

Acceso al PC

  1. Acceso al PC Cliente: Haz doble clic en el PC que actuará como cliente PPPoE.

Configuración de PPPoE

  1. Establecimiento de la Conexión PPPoE:
    • Entra en "Desktop" > "PPPoE Dialer".
    • Introduce el nombre de usuario y contraseña del servidor PPPoE.
    • Pulsa "Connect" para iniciar la conexión.

Verificación de la Conexión

  1. Comprobación del Estado de Conexión:
    • Abre el "Command Prompt".
    • Usa ipconfig para verificar la asignación de IP.
    • Emplea ping para probar la conectividad.

Configuración en Router Cliente

Configuración en un Router 4331
  1. Acceso al Router Cliente: Haz doble clic en el router cliente y abre la CLI.
Configuración del Cliente PPPoE
  1. Perfil del Cliente PPPoE: Define el perfil PPPoE y configura la autenticación utilizando el protocolo PAP (Password Authentication Protocol). Sigue estos pasos en la CLI del Router:
  • Configurar la interfaz Ethernet para PPPoE:
enable
conf t
interface GigabitEthernet0/0/0
no shutdown
no ip address
no ip proxy-arp
pppoe enable
pppoe-client dial-pool-number 1
duplex auto
speed auto
exit

Foto resultado Final:

  • Configurar la interfaz Dialer para gestionar la conexión PPPoE:
interface Dialer1
dialer pool 1
ip address negotiated
encapsulation ppp
ppp authentication pap
ppp pap sent-username [username] password [password]
exit

  • [username] y [password] corresponden a tus credenciales del servidor PPPoE.
    Foto del resultado:

  • Establecer la ruta por defecto a través de la interfaz Dialer:

ip route 0.0.0.0 0.0.0.0 Dialer1

Foto resultado final:

Verificación de la Conexión

  1. Comprobación del Estado de Conexión:

    • Usa el comando show pppoe session para verificar la sesión PPPoE. (Puedes usarlos en varios router)

    • Usa el comando Ping [Puerta de enlace del servidor PPPOE] para verificar que la sesión PPPoE este activa y funcional.

    • Puedes utilizar Tambien show ip interface brief para ver las interfaces activa del router..

2. Configuración de un Router WR300N en Cisco Packet Tracer para Conexión PPPoE

La configuración de un router WR300N en Cisco Packet Tracer para usar una conexión PPPoE implica varios pasos detallados. Este proceso es crucial para establecer una conexión a Internet segura y eficiente utilizando el protocolo PPPoE, que es comúnmente utilizado por proveedores de servicios de Internet, especialmente en conexiones de banda ancha.

Paso 1: Acceso al Router WR300N

  • Haga doble clic en el ícono del router para acceder a su interfaz gráfica de usuario (GUI).

Paso 2: Navegación en la GUI

  • En la ventana de la GUI del router, encontrará diferentes opciones para configurar el dispositivo.

Paso 3: Configuración del Tipo de Conexión a Internet

  • Busque el apartado llamado 'Internet Connection Type'
  • Cambie el tipo de conexión de "DHCP "a 'PPPoE'. Esta opción permite que el router utilice el protocolo PPPoE para la autenticación y configuración de la conexión a Internet.

Paso 4: Ingreso de Credenciales PPPoE

  • Una vez seleccionado PPPoE como tipo de conexión, se le pedirá que ingrese credenciales. Estas suelen ser proporcionadas por su proveedor de servicios de Internet.
  • Ingrese el 'Username' (Nombre de Usuario) y 'Password' (Contraseña) en los campos correspondientes. Asegúrese de ingresar estos datos correctamente, ya que son esenciales para la autenticación y el acceso a la red.

Paso 5: Guardar y Aplicar Cambios

  • Después de ingresar las credenciales, busque un botón para guardar o aplicar los cambios. Esto esta etiquetado a bajo del todo de la pagina del router en un boton de que poner "Save settings".

  • Haga clic en este botón para confirmar y posteriormente se procedera a levantar la configuración PPPoE en el router.

    Paso 6: Verificación de la Configuración

  • Para verificar que la configuración es correcta y que el router se ha conectado a Internet mediante PPPoE, puede usar herramientas de diagnóstico disponibles en el router, como el comando 'ping' para probar la conectividad.

  • También puede revisar el estado de la conexión en la GUI para asegurarse de que la conexión PPPoE esté activa y funcional.

  • NOTA IMPORTANTE: Este router trae una NAT simple y DHCP por defecto.
