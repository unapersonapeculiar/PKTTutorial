NAT

Manual de Configuración NAT en un Router Cisco

Introducción

Este manual detalla los pasos para configurar NAT (Network Address Translation) en un router Cisco. NAT permite que múltiples dispositivos de una red local (LAN) compartan una única dirección IP pública para acceder a Internet o otras redes. También permite la asignación de puertos específicos para servicios internos para convertirlos en servicios externos.

Materiales Necesarios

  • Un router Cisco con interfaces FastEthernet o GigabitEthenet.
  • Cables de red para conectar el router a la red interna y externa.
  • Dirección IP pública para la interfaz externa.
  • Direcciones IP privadas para la red interna.

Pasos de Configuración

1. Acceso al Router
  1. Conectarse al router a través del CLI.
  2. Entre al modo privilegiado: 
    Router>enable
  3. Entre al modo de configuración global: 
    Router#configure terminal
2. Configuración de Interfaces
Interfaz Externa (Fa0/0)
  1. Seleccionamos la interfaz FastEthernet 0/0 
    Router(config)#interface Fa0/0
  2. Active la interfaz: 
    Router(config-if)#no shutdown
  3. Asigne una dirección IP y máscara de subred: 
    Router(config-if)#ip address 10.0.0.1 255.0.0.0
  4. Configure la interfaz como externa para NAT: 
    Router(config-if)#ip nat outside
Interfaz Interna (Fa0/1)
  1. Seleccione la interfaz FastEthernet 0/1: 
    Router(config)#interface Fa0/1
  2. Active la interfaz: 
    Router(config-if)#no shutdown
  3. Asigne una dirección IP y máscara de subred: 
    Router(config-if)#ip address 192.168.1.1 255.255.255.0
  4. Configure la interfaz como interna para NAT: 
    Router(config-if)#ip nat inside
3. Configuración de Lista de Acceso para NAT
  1. Creamos una lista de acceso para permitir las direcciones IP internas que usarán NAT: 
    Router(config)#access-list 99 permit 192.168.1.0 0.0.0.255
4. Configuración de NAT
Caso 1: Traducción uno a uno
  1. Definimos un pool de direcciones IP para NAT: 
    Router(config)#ip nat pool CONJUNTO1 10.0.0.2 10.0.0.255 netmask 0.255.255.255
  2. Configuramos NAT para usar la lista de acceso y el pool: 
    Router(config)#ip nat inside source list 99 pool CONJUNTO1
Caso 2: Traducción con Sobrecarga (PAT)
  1. Defina un pool con una sola dirección IP: 
    Router(config)#ip nat pool CONJUNTO1 10.0.0.1 10.0.0.1 netmask 0.255.255.255
  2. Configure NAT para usar la lista de acceso y el pool con sobrecarga: 
    Router(config)#ip nat inside source list 99 pool CONJUNTO1 overload
5. Asignación de Puertos
  1. Asignamos el puerto 80 de la IP interna 192.168.1.3 al puerto 80 de la IP externa 10.0.0.1: 
    Router(config)#ip nat inside source static tcp 192.168.1.3 80 10.0.0.1 80
6. Configuración de la Ruta por Defecto
  1. Configure la ruta por defecto hacia la interfaz externa: 
    Router(config)#ip route 0.0.0.0 0.0.0.0 Fa0/0

Verificación del NAT

  1. Para verificar la configuración de NAT y las traducciones actuales, usamos el siguiente comando: 
    Router#show ip nat translations
  2. Este comando muestra una tabla con las traducciones NAT actuales, incluyendo las direcciones IP internas y externas y los puertos asociados.
